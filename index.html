Projects

Spider Image Classifier

This project aims to develop a Convolutional Neural Network (CNN) trained to classify images from 15 distinct spider species, with the primary goal of determining whether a spider image raises medical concerns. To enhance the model's robustness and performance, the project delves into data augmentation techniques. Additionally, the project's secondary objective involves the deployment of an API that empowers users to swiftly identify spiders that may pose medical risks.

Tutorial for Titanic Dataset

This project presents one of the many approaches to address one of Kaggle's most popular challenge, based on certain features, one must predict whether a passenger will survive this catastrophe. I review the most commonly used machine learning algorithms for classification, including ensemble methods. While some algorithms showed promise, the final accuracy score of 83% was achieved using a Random Forest algorithm, considering cross-validation and hyperparameter tuning.
